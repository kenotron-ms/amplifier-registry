How Ratings Work
Transparent, community-sourced quality signals
19

Team Members Swept

Public GitHub repos are swept nightly for new bundles, changes, and usage signals. Only repos with public visibility are listed. Results update automatically.

AI

LLM-as-Judge Ratings

An automated pipeline evaluates each bundle on documentation quality, capability breadth, prompt design, and observable output quality. Runs on every update and bi-weekly for unchanged bundles.

∞

Community Signals

Users self-report which bundles they use. Mentions in GitHub Discussions, Teams, and Slack are picked up by the sweep. Every report increments the weekly user count.

Built something with Amplifier?

Share your bundle with the community. We'll run an LLM quality check, seed usage tracking, and add it to the registry. Open-source and team-private bundles both welcome.
